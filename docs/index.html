Info Server

L'utilité de ce composent coder en PHP, est de récupère les informations vitales comme la capaciter mémoire avec la consommation de celle-ci, la capacité CPU avec ça consommation. La portabilité dans diffèrent projet est primordiale, avec composer l'installation et la mise en place est un jeu d'enfant.
