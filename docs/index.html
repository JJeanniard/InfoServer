Info Server

Description

L'utilité de ce composent codé en PHP, est de récupère les informations vitales sur un serveur(Ubuntu ou Debian), via à une connexion ssh qui doit être définie par avance et implémenté dans l'instance de la classe. Comme exemple: la capacité mémoire avec la consommation de celle-ci, la capacité CPU avec ça consommation. La portabilité dans diffèrent projet est primordiale, avec composer l'installation et la mise en place est un jeu d'enfant.

Les prérequis est d'avoir composer installé sur son poste ou l'utiliser en standalone(composer.phar), de connaitre un minimum sa gestion sur les packages.
Lien officiel de Composer --> Download

Les méthodes

Les méthodes proposé sont:

  • getSysteme()
  • getCpu()
  • getRam()
  • getSawp()
  • getLoadAverage()
  • getReseau()
  • getDisk()
  • getUptime()

getSysteme()

La méthode getSysteme() retourne un tableau, sur les informations sur :

  • Le hostname
  • L'os
  • La date du jour et l'heure (en fonction du fuseau horaire réglé sur le serveur)
  • La version du kernel
  • L'architecture du systéme
  • L'uptime (est le nombre de jour que le serveur fonctionne sans avoir redémarrer)

getCpu()

  • Le marque du processeur
  • Le nombre de coeur
  • L'usage du processeur en pourcentage
